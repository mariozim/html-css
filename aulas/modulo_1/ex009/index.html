tipos de listas

lista ordenada

listas onde a ordem faz diferença

usamos a tag "ol(ordened list)" pra criar a lista e a tag "li(list item)" para criar um objeto da lista ex:

  1. acordar
  2. tomar cafe da manha
  3. reerguer a união sovietica
  4. tomar banho

lista não ordenada

é ultilizado quando a ordem dos fatores da lista não faz diferença, portanto não serão numerados

seguindo a logica de nomeação anterior temos a lista não ordenada"ul(unnorded list) pra criar a lista e a tag "li(list item)" para criar um objeto da lista ex:

  • leite
  • pão
  • manteiga
  • café

juntando listas

juntar listas é um conceito basico, basta visualizar o formato que suas listas devem estar e reproduzir essa formatação em html

ex:

minhas comidas favoritas

  1. comidas
    • lasanha
    • churrasco
    • pizza
  2. bebidas
    • refrigerante
    • suco
    • água
