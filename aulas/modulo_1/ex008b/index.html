citação simples

podemos marcar uma citação como no exemplo:

Como diria o pai de um amigo,o computador é um burro muito rapido


podemos marcar uma citação com aspas simples 'assim' ou aspas duplas assim

Como diria o 'pai de um amigo',o computador é um burro muito rapido


ou um jeito mais formal com a tag 'q' assim

Como diria o pai de um amigo,o computador é um burro muito rapido

citação completa

podemos marcar uma citação completa com a tag 'blockquote' assim

Strings in Python are arrays of bytes representing Unicode characters. However, Python does not have a character data type, a single character is simply a string with a length of 1.
